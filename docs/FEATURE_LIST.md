## User Stories Summary

### Buyer
1. Sebagai pembeli, saya ingin mencari produk agar saya bisa menemukan yang saya butuhkan
2. Sebagai pembeli, saya ingin melihat detail produk agar saya bisa memutuskan untuk membeli
3. Sebagai pembeli, saya ingin menambah ke keranjang agar saya bisa checkout nanti
4. Sebagai pembeli, saya ingin checkout agar saya bisa memesan produk
5. Sebagai pembeli, saya ingin hubungi seller via WhatsApp agar saya bisa konfirmasi

### Seller
1. Sebagai penjual, saya ingin tambah produk agar saya bisa menjualnya
2. Sebagai penjual, saya ingin edit produk agar saya bisa mengupdate informasi
3. Sebagai penjual, saya ingin lihat order agar saya bisa memprosesnya
4. Sebagai penjual, saya ingin update status agar pembeli tahu perkembangan

### Admin
1. Sebagai admin, saya ingin verifikasi UMKM agar platform tetap terpercaya
2. Sebagai admin, saya ingin kelola produk agar tidak ada produk yang melanggar
3. Sebagai admin, saya ingin monitoring agar saya bisa melihat aktivitas
